# ✨ Smoothify

Smoothify is a modern image enhancement web app that smoothens skin, reduces blemishes, and applies real-time filters — all wrapped in a clean, aesthetic UI.

---

## 🚀 Features

* 🎨 Multiple filters: Smooth, Gaussian, Median (B/W), Blur, Sharpen
* 💆 Smart skin smoothing (blemish reduction without heavy blur)
* 🎚 Adjustable intensity control
* 🖼 Before & After comparison
* 📥 Download processed images
* 🔐 Login & Signup system (basic authentication)
* 🌸 Custom pink aesthetic UI

---

## 🛠 Tech Stack

* Python
* Streamlit
* OpenCV
* NumPy

---

## ⚙️ Run Locally

```bash
git clone https://github.com/qnoushka/smoothify.git
cd smoothify
pip install -r requirements.txt
streamlit run app.py
```
