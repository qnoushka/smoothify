# ✨ Smoothify

A sleek AI-powered image enhancement app built with Streamlit that smoothens skin, removes blemishes, and applies real-time filters — all wrapped in a modern aesthetic UI 💅

---

## 🚀 Features

- 🎨 Multiple filters (Smooth, Gaussian, Median, Blur, Sharpen)
- 💆‍♀️ Smart skin smoothing (blemish reduction without heavy blur)
- 🎚 Adjustable intensity slider
- 🖼 Before & After comparison view
- 📥 Download processed images
- 🔐 Login & Signup system (basic authentication)
- 🌸 Custom pink aesthetic UI

---

## 🛠 Tech Stack

- Python
- Streamlit
- OpenCV
- NumPy
